# Senior Executive Trainee (SET) - Telecom Syllabus

## Part-I: Common Aptitude Test
**Duration:** 180 Minutes (combined with Part-II)  
**Questions:** 40 | **Marks:** 40

### Topics:
1. **Quantitative Ability and Data Sufficiency**
2. **Reasoning** (analytical, logical, and critical reasoning)
3. **Verbal Ability, Reading Comprehension, and Analysis**

---

## Part-II: Core Technical - Telecom
**Questions:** 160 | **Marks:** 160  
**Total Duration:** 180 Minutes (combined with Part-I)

### 1. Basic Signal Processing (15% weightage)
- Modulation and Demodulation
- Multiplexing Techniques
- Spectrum Management
- Telecommunications Technologies Concepts

### 2. Wireless Telecommunications Technologies (15% weightage)
- Wireless Communication Principles
- Mobile Antenna System (Basics, Azimuth, Tilt, VSWR)
- Fundamentals of GSM (2G/3G)
- Fundamentals of LTE
- Fundamentals of IP Multimedia Subsystem (IMS)
- Fundamentals of 5G and Industrial Applications (5G Use Cases and Captive Networks)
- KPIs of 2G/3G/4G/5G Mobile Networks (voice & data quality)
- Satellite Communications (LEO, MEO)
- Fundamentals of Telco-Cloud
- Mobile Radio Back Haul (M/W, UBR, E-Band)
- Spectrum Management
- Fixed Wireless Access
- Fundamentals of USIM/ESIM/ISIM and Network Access Security
- Fundamentals of Network Management System (NMS)
- Standardization in Wireless Communication

### 3. Wireline Telecommunications Technologies through ICT (15% weightage)
- Fundamental of Signaling (SS7/SIGTRAN) & Various Protocols (SIP, Megaco, H.248, MGCP, RTP, RTCP)
- Disaster Management & Mitigation
- NGN Switches, Soft Switches
- Telecommunication Call Routing and Switching Concepts
- Broadband Internet Technologies
- FTTx Technology
- Signaling in PSTN and SSTP
- C4 & C5 NGN Architecture and Call Flow
- NGN Protocols (SIP/H.248/RTP/SIGTRAN)
- IMS Network Architecture
- ISDN (PRI/BRI), EPABX, SIP Services
- Wi-Fi Services and Encrypted Hotspot, Enterprise Hotspot Solutions

### 4. Telecom Transmission Infrastructure (15% weightage)
- Fundamentals of Transmission Systems
- Optical Fiber Technology
- Optical Fiber Systems (DWDM, OTN, FTTH, GPON, EPON)

### 5. IP/Data Networks (15% weightage)
- Network Elements
- Routing Concepts and Protocols
- OSI and TCP/IP Model
- Caching and Peering
- Secured IP Network
- Fundamental Concept of IPv4 Addressing, Subnetting, Supernetting
- IPv6 Addressing: Scheme, Types of IPv6 Addresses, IPv6 Routing, Tunneling
- Fundamentals of MPLS
- Network Design for Next Generation IP Cloud Network
- IP Network QoS
- Proxy Services, DNS Services
- Cyber Security and Firewall

### 6. Emerging Technologies (15% weightage)
- Fundamentals of Internet of Things (IoT) and M2M
- Blockchain
- Software Defined Networks
- Cloud Computing and Data Centers
- Artificial Intelligence and Telecom Network

### 7. Project Management (10% weightage)
- Introduction to Project Management and Project Selection
- Project Planning and Implementation
- Project Management Tools: Bar Chart, Gantt Chart, PERT, CPM
